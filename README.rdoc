== Задания на вакансию RoR программиста Aviasales
http://www.aviasales.ru/content/rubyror-kaban

===Описание: сервис производит поиск перелётов из одного аэропорта в другой. Возможны как прямые перелёты так и пересадки(до 2-х). Также возможно продолжение маршрута. Селекция результатов (яваскриптом) по общему времени полёта, цене, времени ожидания между пересадками, количеству пересадок. Данные о единичных перелётах и кешированные результаты поиска хранятся в Redis DB.

===Требования:
1. Должен быть установлен гем redis ~> 3.0.2
2. Запущен сервер redis на 127.0.0.1:6379
3. Перед тестами заполнить базу данных значениями, запустив скрипт /scripts/create_track_database.rb
